# Rich Text Editor

Location: `components/rich-text-editor/`

Built with Lexical editor.

## Custom Nodes

- `ImageNode`
- `VideoNode`
- `CustomCodeHighlightNode`

## Plugins

- Code highlighting (`@lexical/code-shiki`)
- Media upload
- Link handling

## Usage

Exports Markdown via `getMarkdown()` method on editor ref.
